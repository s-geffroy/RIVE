# 2. Le nom RIVE et son architecture institutionnelle

## 2.1 Nom public

**RIVE — Réseau Intégré de Vie Éducative**

Le sigle est volontairement simple. Il doit pouvoir être prononcé par une famille, écrit sur un bâtiment, utilisé dans un document administratif et compris sans explication interminable.

Chaque mot compte.

- **Réseau** : RIVE n’est pas nécessairement un bâtiment unique. C’est une organisation de bassin.
- **Intégré** : les fonctions éducatives ne sont pas juxtaposées ; elles sont coordonnées.
- **Vie** : le projet dépasse la classe et inclut les rythmes, les repas, le sport, la culture, les vacances et le repos.
- **Éducative** : l’objectif reste le développement de l’enfant, pas le simple gardiennage.

## 2.2 Niveaux de nommage

| Niveau | Nom |
|---|---|
| Système public | RIVE |
| Nom développé | Réseau Intégré de Vie Éducative |
| Nom institutionnel | Service public RIVE |
| Architecture locale | Campus-réseau éducatif intégré RIVE |
| Unité locale | RIVE + nom du territoire |
| Sites d’âge | Maisons RIVE |
| Activités longues | Parcours RIVE |
| Document de référence | Blue Book RIVE |

Exemples : **RIVE Montauban Sud**, **RIVE Toulouse Nord**, **RIVE Castres Bassin**.

## 2.3 Ce que RIVE n’est pas

RIVE n’est pas une école totale.  
RIVE n’est pas une solution miracle.  
RIVE n’est pas un système destiné à garder les enfants le plus longtemps possible hors de leur famille.  
RIVE n’est pas une privatisation de l’éducation.  
RIVE n’est pas une plateforme IA déguisée en école.  
RIVE n’est pas une structure réservée aux enfants en difficulté.

Le projet est plus exigeant : créer un service public intégré qui améliore la qualité éducative ordinaire, tout en étant assez robuste pour protéger les plus vulnérables.
