# RIVE — Blue Book V1 (révision v1.3)

## Index des chapitres

**RIVE** signifie **Réseau Intégré de Vie Éducative**.

Ce dossier contient le Blue Book RIVE découpé en chapitres, avec un fichier Markdown par chapitre.

### Trois entrées documentaires

Selon le temps disponible :

- 📄 [**Brief exécutif (1 page)**](./EXEC_BRIEF.md) — lecture 2 min, pour ministre / président d’EPCI / décideur.
- 📑 [**Dossier politique court (5 pages)**](./DOSSIER_POLITIQUE.md) — lecture 15 min, pour cabinet / DGS / chef de bureau.
- 📚 **Blue Book complet** (ci-dessous, 27 chapitres) — référence opérationnelle complète.
